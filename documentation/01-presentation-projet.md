# 01 – Présentation du projet

## Nom du projet
Bienvenue à Angoulême – Le Blog

## Contexte
Ce projet a été réalisé dans le cadre de la formation DWWM (Développeur Web et Web Mobile).

L’objectif est de concevoir un blog complet disposant :
- d’une interface publique
- d’un espace administrateur
- d’une gestion des articles, catégories, utilisateurs et commentaires
- d’un système d’authentification sécurisé

## Objectifs pédagogiques

- Mettre en œuvre une architecture MVC personnalisée
- Concevoir une base de données relationnelle
- Sécuriser les interactions utilisateur
- Préparer le projet à un déploiement réel
- Versionner le code via GitHub

## Public cible

- Habitants d’Angoulême
- Touristes
- Passionnés d’actualité locale
