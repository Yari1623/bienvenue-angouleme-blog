# 04 – Base de données

Le projet utilise une base relationnelle MySQL.

## Tables principales

- users
- posts
- categories
- comments
- events
- likes

## Relations

- Un utilisateur peut créer plusieurs articles
- Un article appartient à une catégorie
- Un article peut recevoir plusieurs commentaires
- Un utilisateur peut aimer plusieurs articles

Un schéma relationnel sera fourni dans le dossier final.
