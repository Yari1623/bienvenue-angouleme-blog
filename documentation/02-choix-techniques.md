# 02 – Choix techniques

## Langage backend
PHP 8.3.14

Justification :
- Langage utilisé dans le cadre de la formation et du stage
- Compatible avec la majorité des hébergements mutualisés
- Stable et performant

## Base de données
MySQL via phpMyAdmin

Justification :
- Base relationnelle adaptée aux relations (articles, catégories, utilisateurs, commentaires)
- Large compatibilité hébergement

## Serveur local
WAMP sous Windows 11

Justification :
- Environnement simple à configurer
- Compatible Apache / MySQL / PHP

## Architecture
MVC personnalisé

Justification :
- Séparation des responsabilités
- Maintenabilité
- Évolutivité
- Meilleure organisation du code

## Front-end
Tailwind CSS (CDN)

Justification :
- Rapidité de développement
- Cohérence visuelle
- Pas de dépendance Node pour un projet de certification

## Graphiques
Chart.js

Justification :
- Visualisation dynamique des statistiques
- Léger et simple à intégrer
