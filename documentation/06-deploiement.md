# 06 – Déploiement

Le projet est conçu pour être déployé sur :

- Hébergement mutualisé Apache
- Serveur PHP 8+
- Base MySQL

Structure prévue :

- public/ comme racine du site
- fichier .env pour configuration
- désactivation du mode debug en production

Procédure d’installation :

1. Importer la base de données
2. Configurer le fichier .env
3. Définir public/ comme document root
